# Depth Map for Technical Readers

This public repository keeps the surface simple, but the underlying system is layered.

## Core thesis

Multi-LLM systems need more than prompts, memory, and message history. They need explicit semantic state, protocol-level integrity, governance rules that can be inspected, runtime gates that can stop or require review, handoff records that survive model changes, and semantic maps for large artifact universes.

## System chain

```text
VCE / Semantic State
  → SIC-JS state format
  → SIC/T protocol invariants
  → Babel constraints + AGR governance rules
  → Integrity / Encoding / Runtime Gates
  → IDDP / SLT multi-agent handoff
  → Semantic Mapping Governance
```

## Where the depth lives

This A repository is the front door. AN∞Node contains formal protocol notes, SIC-JS schema and examples, AGR governance runtime surfaces, IDDP / SLT handoff verification materials, semantic mapping methodology, public-safe evidence objects, implementation and test reports, claim boundaries, and evidence matrices.

## What this repository proves

It proves that the work has a coherent public shape. It does not prove external validation, production safety, or universal applicability.

# Exhibit Relation Map / 展示關係圖

The five exhibits are not five separate projects. They are five views of one system.
五個展示不是五個獨立的項目。它們是同一系統的五個視角。

```text
[1] Semantic State Infrastructure / 語義狀態基礎設施
        produces state objects / 產生狀態物件
            ↓
[2] Semantic Integrity Protocols / 語義完整性協議
        protect state during handoff / 在交接過程中保護狀態
            ↓
[3] Runtime Governance / 運行治理
        turns constraints into runtime decisions / 將約束轉化為運行決策
            ↓
[4] Multi-LLM Coordination / 多模型協調
        moves work across agents without losing state / 在代理間移動工作而不失去狀態
            ↓
[5] Semantic Mapping Governance / 語義映射治理
        keeps the larger artifact universe navigable and governable / 保持更大的構件宇宙可導航且可治理
```

## What this map does not claim / 本圖不聲稱的內容

This public entry repository does not claim full production safety, external validation, or complete AGI governance.
本公開入口倉庫不聲稱完整的生產安全性、外部驗證或完整的 AGI 治理。

It claims a coherent technical direction / 它聲稱一個連貫的技術方向:

> semantic state + protocol integrity + AGR governance + runtime gates + multi-model handoff + semantic mapping.
> 語義狀態 + 協議完整性 + AGR 治理 + 運行閘門 + 多模型交接 + 語義映射。

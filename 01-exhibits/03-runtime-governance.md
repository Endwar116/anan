# Exhibit 03 — Runtime Governance

## Simple layer

A rule written in a document is not enough.

The system must be able to do something with the rule while it is running.

For example:

- continue
- stop
- ask for review
- hand off
- close the task

## Technical layer

This is where governance becomes behavior.

- Integrity Gate
- Encoding Gate
- SICKernel

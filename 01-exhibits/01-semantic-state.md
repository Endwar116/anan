# Exhibit 01 — Semantic State

## Simple layer

When a person continues another person's work, they need more than the last sentence.

They need to know:

- what the goal was
- what rules were already agreed on
- what has already failed
- what should not be changed
- what still needs review

AI systems have the same problem.

## Technical layer

This line treats understanding as something that can be externalized into state objects.

Related deep systems:

- VCE — semantic coordinate framing
- ACV — continuity vector concept
- state graph / lineage
- state verification
- state re-entry

## Boundary

This does not claim AI consciousness or perfect state preservation. It claims that operational semantic state can be represented and checked in useful ways.

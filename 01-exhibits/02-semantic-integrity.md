# Exhibit 02 — Semantic Integrity

## Simple layer

If one AI hands work to another AI, the next AI should not just read the old notes.

It should inherit the meaning of the work.

That means it should know:

- the current state
- the intent
- the important events
- the boundaries
- the reason for stopping or continuing

## Technical layer

This is the protocol layer.

Related deep systems:

- SIC/T — semantic integrity and transfer protocol
- SIC-JS — state format
- protocol lineage
- handoff state
- fail-closed behavior

- complete AGI governance / 完整 AGI 治理
- universal semantic alignment / 通用語義對齊
- production safety guarantees / 生產安全保證
- external scientific validation / 外部科學驗證

It claims / 它聲稱:

- active technical system design / 活躍的技術系統設計
- public-safe proof surfaces / 公開安全的證明介面
- protocol-oriented research / 協議導向的研究
- clear public/private boundaries / 清晰的公開/私有邊界

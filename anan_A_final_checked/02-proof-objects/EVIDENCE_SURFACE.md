# Evidence Surface

This repo uses small public-safe proof objects.

They are not full production systems. They are inspection windows.

## What counts as evidence here

| Evidence type | What it proves |
|---|---|
| State sample | The system has a concrete state surface. |
| Gate trace | A rule can become a runtime decision. |
| Routing trace | Multi-agent handoff can be represented. |
| Mapping sample | A messy archive can become a governed semantic map. |
| AGR trace | A governance rule can change item status and leave proof. |

## What it does not prove

These files do not prove universal safety, complete automation, external replication, or production readiness.

They prove that the work has inspectable surfaces and clear boundaries.

# One-Page Magazine Spread / 一頁雜誌版面

## AnAn builds the layer underneath AI applications.
## AnAn 建造 AI 應用底層的基礎設施。

Not another chatbot. Not another wrapper.
不是另一個聊天機器人。不是另一個包裝器。

A state, protocol, and governance layer for multi-LLM systems.
這是多 LLM 系統的狀態、協議與治理層。

---

### The problem / 問題

AI systems can read instructions, but they often lose operational state when a task moves across sessions, agents, or models.

AI 系統能讀取指令，但當任務在對話、代理或模型之間轉移時，它們往往失去操作狀態。

---

### The response / 回應

A public-safe technical body of work around / 圍繞以下方向的公開安全技術工作:

- semantic state / 語義狀態
- protocol handoff / 協議交接
- runtime governance / 運行治理
- multi-agent coordination / 多代理協調
- semantic mapping / 語義映射
- AGR governance rules / AGR 治理規則

---

### The signature idea / 標誌性概念

A handoff should not be a vibe.
交接不應該只是感覺。

It should be a state package with intent, constraints, event history, boundary, and verification.
它應該是一個包含意圖、約束、事件歷史、邊界與驗證的狀態包。

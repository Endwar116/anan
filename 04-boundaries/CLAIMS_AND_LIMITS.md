# Claims and Limits

## What this public repo can support

| Claim | Status |
|---|---|
| AnAn works on semantic state and coordination infrastructure. | PUBLIC_POSITIONING |
| The repo contains public-safe proof surfaces. | INSPECTABLE_SAMPLE |
| SIC-JS can be represented as a JSON state surface. | PUBLIC_SAFE_DESIGN |
| AGR can be shown as a governance decision trace. | PUBLIC_SAFE_SAMPLE |
| Semantic Mapping can show raw → classified → governed transformation. | PUBLIC_SAFE_SAMPLE |

## What this public repo does not prove

| Not claimed | Why |
|---|---|
| Complete external validation | Requires external replication. |
| Production safety guarantee | Requires deployment-specific evaluation. |
| Universal semantic theory | Belongs to deeper research and is not asserted here. |
| Full automation of governance | Public samples show surfaces, not full production behavior. |
